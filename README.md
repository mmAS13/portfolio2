# Professional Web Developer Portfolio (2023 Practice)
A modern, dark-themed personal portfolio website featuring smooth navigation and interactive UI elements. This project was developed in 2023 as a dedicated practice exercise to master frontend development, responsive design, and DOM manipulation.

# Key Features
Dynamic Sticky Header: The navigation bar automatically transitions to a "sticky" state after scrolling 120px for better UX.

Mobile-Responsive Menu: A functional hamburger menu built with JavaScript for seamless browsing on smaller screens.

Interactive Portfolio Grid: Includes hover-effects and GitHub integration for project showcases.

Advanced Styling: Built using CSS variables (root), Flexbox, and Grid for a clean, maintainable layout.

Smooth Navigation: Internal linking with smooth-scroll behavior for a premium feel.

# Tech Stack
HTML5: Semantic structure and layout.

CSS3: Custom animations, media queries, and dark-mode aesthetics.

JavaScript (ES6): Logic for scroll events and menu toggles.

Icons: Font Awesome for social and service icons.

Typography: Inter font for a modern look.

# File Structure

# Project Context
This was a practice-only project created in 2023 to bridge the gap between basic coding and real-world web applications. It focuses on clean code structure and understanding how different frontend technologies work together.
